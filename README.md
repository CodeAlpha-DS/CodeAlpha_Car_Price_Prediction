# CodeAlpha_Car_Price_Prediction

CodeAlpha Data Science Internship — **Task 3: Car Price Prediction with Machine Learning**

## Dataset (Official)

**Kaggle:** https://www.kaggle.com/datasets/vijayaadithyanvg/car-price-predictionused-cars

File: `data/car data.csv` — 301 used car listings with selling price, present price, kms driven, fuel type, etc.

To re-download from Kaggle (requires [Kaggle API setup](https://www.kaggle.com/docs/api)):

```bash
pip install kaggle
python download_data.py
```

## Setup

```bash
pip install -r requirements.txt
python main.py
```
